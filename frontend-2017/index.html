  • Уменьшение количества HTTP-запросов
  • Использование CDN
  • Использование HTTP-заголовка Expires
  • Размещение CSS в HEADER
  • Размещение скриптов в конец страницы

  • Минификация файлов
  • Исключение (по возможности) редиректов
  • Исключение повторяющегося кода
  • Кэширование все, что можно кэшировать

  • Lazy load
  • Серверный рендеринг
